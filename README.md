# Multiple Databases with docker-compose
- mysql
- mariadb
- postgres
- couchdb
- redis

# GUI
- phpmyadmin
- pgadmin4
- adminer
- fauxton for couchdb gui http://127.0.0.1:5984/_utils
- redis-commander

## How to use
clone this project then open a terminal in this path and run the below command
- docker-compose up --build -d
